# Student_Register_django_project

## Description
The Student Register Management System is a web-based application developed using Django.  
It allows users to register students, view records, update details, and delete student information in an organized way.

## Features
- Student registration form
- View student records in table format
- Update student details
- Delete student records
- User-friendly interface

## Technologies Used
- Python
- Django
- HTML
- CSS
- Bootstrap
- SQLite (default Django database)
